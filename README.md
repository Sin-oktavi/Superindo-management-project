# Superindo Management Project

## Overview
This project focuses on designing an integrated database system for PT Superindo, one of Indonesia’s major retail supermarket companies. The project was developed as a team-based academic project to address data management challenges across key business operations such as customer transactions, product inventory, membership, and feedback management.

## Business Background
PT Superindo operates multiple retail stores and handles large volumes of daily transaction and customer data. As business operations grow, the company requires a structured database system to ensure data consistency, efficient transaction recording, stock tracking, and better support for decision-making.

## Project Objective
The objective of this project was to design a relational database system that can support PT Superindo’s operational processes by integrating core business data into a structured and scalable system.

## Scope of Work
The project covered:
- Business process and problem analysis
- Database requirement identification
- Entity Relationship Diagram (ERD) design
- Relational database schema design
- Database normalization
- SQL-based database implementation concept
- Recommendations for future BI/AI development

## Proposed Solution
Our team designed an integrated database system that manages:
- Customer data
- Product and stock information
- Transaction records
- Membership program data
- Staff information
- Customer feedback

The solution was aimed at improving data organization, reducing redundancy, and creating a stronger foundation for reporting, analytics, and future digital transformation initiatives.
